MotionPy

A motion detection system using a Raspberry Pi 3, camera module, breadboard, some paper clips and a top-of-the-line cardboard box. A Python server (surprise surprise, it's Flask again) running on the Raspberry Pi sends an Android app, written in Java and Kotlin, a notification and a picture when movement is detected. Images are saved online via Firebase.
